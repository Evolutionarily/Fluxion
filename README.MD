# Fluxion

Fluxion is a python Coloring library and is very simple to use

this library will be updated in the future


# Install
```bash
pip install Fluxion
```

# Examples

```py
from Fluxion import Colors

print(f"{Colors.Blue}Hello World!")
#will print out hello world in blue.
```
